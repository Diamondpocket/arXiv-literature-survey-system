# 检索增强生成(RAG)周报

## 本周概览 Weekly Overview

本周检索增强生成(RAG)领域共发表127篇论文，证据基础规模显著增大。研究呈现出明显的多元化趋势，从通用检索管道向专业变体发展，包括评测基准、领域适配、结构化知识、多模态设置和可靠性导向设计等方向。方法层面呈现模块化趋势，检索器、重排序器、生成器、评估器等组件被视为独立的优化目标。

## 分类体系摘要 Taxonomy Summary

本周研究主要分布在以下7个类别：

1. **评测、基准与诊断 (Evaluation, Benchmarks, and Diagnostics)** - 14篇论文
   - 代表性工作包括MathNet数学推理与检索基准、FRESCO动态语义冲突重排序器优化框架
   - 主要创新类型：基准评测(9篇)、系统架构(3篇)

2. **核心检索与排序方法 (Core Retrieval and Ranking Methods)** - 35篇论文
   - 代表性工作包括WorldDB向量图记忆引擎、NaviRAG主动知识导航框架
   - 主要创新类型：系统架构(15篇)、检索方法(12篇)、生成方法(8篇)

3. **领域特定RAG应用 (Domain-Specific RAG Applications)** - 21篇论文
   - 代表性工作包括金融QA架构比较研究、漏洞分析RAVEN系统
   - 主要创新类型：系统架构(12篇)、检索方法(5篇)

4. **智能体与多智能体RAG (Agentic and Multi-Agent RAG)** - 21篇论文
   - 代表性工作包括MASS-RAG多智能体合成框架、AutoSearch自适应搜索深度
   - 主要创新类型：系统架构(19篇)、检索方法(2篇)

5. **多模态与视觉RAG (Multimodal and Visual RAG)** - 8篇论文
   - 代表性工作包括AeroRAG航空视觉推理、KIRA知识密集型图像检索
   - 主要创新类型：系统架构(5篇)、检索方法(3篇)

6. **知识图谱与结构化知识RAG (Knowledge Graph and Structured Knowledge RAG)** - 4篇论文
   - 代表性工作包括EvoRAG自进化知识图谱RAG、EHRAG混合超图构造
   - 主要创新类型：检索方法(3篇)、系统架构(1篇)

7. **可靠性、安全、隐私与防护 (Reliability, Safety, Privacy, and Security)** - 11篇论文
   - 代表性工作包括ArbGraph冲突感知证据仲裁、神经符号临床指南冲突解析
   - 主要创新类型：系统架构(7篇)、生成方法(2篇)、评测基准(1篇)

## 方法对比摘要 Method Comparison

本周研究在方法复杂度、场景、优势与劣势方面呈现多样化特征：

1. **高复杂度方法**：
   - MathNet：构建跨47国、17语言的数学问题数据集，提供数学推理和检索基准
   - UniDoc-RL：采用分层动作空间和密集奖励机制的强化学习框架
   - NIM4-ASR：多阶段训练范式优化的LLM ASR框架

2. **中复杂度方法**：
   - MASS-RAG：多智能体合成框架，分离证据总结、提取和推理
   - WorldDB：基于递归世界和不可变节点的记忆引擎
   - Adaptive Query Routing：根据查询复杂度和文档结构动态选择检索策略

3. **场景适应性**：
   - 多模态场景：AeroRAG将航空图像转换为结构化视觉知识
   - 专业领域：EvoRAG通过反馈驱动自进化知识图谱
   - 安全关键：ArbGraph解决长形式RAG中的事实冲突

4. **数据驱动方法**：
   - 105篇论文明确采用数据驱动方法，22篇未明确说明
   - 典型代表包括TeleEmbedBench电信领域嵌入基准、Skill-RAG故障感知检索增强

5. **方法优势**：
   - 性能提升：MathNet上SOTA推理模型性能达78.4%
   - 效率优化：SAGE实现90%令牌减少同时保持准确性
   - 可靠性增强：ArbGraph减少幻觉并提高信息密度

## 趋势分析 Trend Analysis

本周研究呈现以下明显趋势：

1. **从通用到专业**：RAG研究从通用检索管道转向专业变体，包括评测、领域适配、结构化知识、多模态设置和可靠性设计等方向。

2. **方法模块化**：检索器、重排序器、生成器、评估器等组件被视为独立的优化目标，促进了系统的灵活性和可定制性。

3. **评测重要性凸显**：70篇论文关注基准评测，40篇关注评估方法，反映了领域对量化性能和标准化评估的需求日益增长。

4. **多智能体协作**：21篇论文聚焦智能体与多智能体RAG，表明协作式信息处理成为重要研究方向。

5. **领域专业化**：金融、医疗、法律等垂直领域RAG应用研究增多，显示领域适配性的重要性。
