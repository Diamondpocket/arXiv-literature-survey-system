# 本周概览 Weekly Overview

本周检索增强生成（RAG）领域共有307篇论文，其中132篇发表于2026年5月，表明该研究方向的持续活跃性。研究主要集中在核心检索与排序方法（62篇），其次是领域应用型RAG（39篇）和多模态与视觉RAG（15篇）。大多数研究采用系统架构创新（system_or_architecture）类型，高置信度研究占主导地位（281篇），其中258篇为数据驱动研究。

# 分类体系摘要 Taxonomy Summary

1. **核心检索与排序方法 (Core Retrieval and Ranking Methods)**：共94篇研究，关注检索和排序的基本方法。代表性工作包括WorldDB向量记忆引擎、Latent Abstraction for RAG（LAnR）以及Skill-RAG等。该领域创新类型多样，包括系统架构、检索方法和生成方法，但有时检索与生成方法的界限较为模糊。

2. **智能体与多智能体 RAG (Agentic and Multi-Agent RAG)**：共22篇研究，探索基于智能体的RAG方法。代表性工作包括MASS-RAG、AutoSearch和VISOR等。该领域面临单智能体与多智能体界限不清晰的挑战，智能体自主程度差异较大。

3. **知识图谱与结构化知识 RAG (Knowledge Graph and Structured Knowledge RAG)**：共12篇研究，专注于知识图谱和结构化知识的应用。代表性工作包括EvoRAG和EHRAG等。部分结构化知识方法也可归类为核心检索方法，且知识图谱的复杂程度差异显著。

4. **多模态与视觉 RAG (Multimodal and Visual RAG)**：共15篇研究，处理文本和视觉信息的整合。代表性工作包括AeroRAG、KIRA和UniDoc-RL等。"多模态"的定义在不同研究中差异较大，跨模态推理能力参差不齐。

5. **领域应用型 RAG (Domain-Specific RAG Applications)**：共39篇研究，针对特定领域的应用。代表性工作包括金融、医疗、教育等领域的RAG系统。该领域与应用边界的界定存在模糊性，某些应用方法具有泛化潜力。

6. **评测、基准与诊断 (Evaluation, Benchmarks, and Diagnostics)**：共19篇研究，关注RAG系统的评估和诊断方法。代表性工作包括MathNet和TeleEmbedBench等，但抽象信息中评估指标的详细程度有限。

# 方法对比摘要 Method Comparison

从比较表中可以看出，不同类别的RAG方法在复杂度、应用场景和优势方面存在显著差异：

1. **核心检索与排序方法**：如Latent Abstraction for RAG（LAnR）采用隐空间检索，复杂度中等，在多个QA基准上表现优异，实现了编码、检索和生成的统一框架。Skill-RAG通过隐藏状态探测和技能路由实现故障感知检索，适用于开放域问答和复杂推理场景。

2. **智能体与多智能体 RAG**：如MASS-RAG使用多个专业代理进行证据处理和合成，复杂度高，通过多代理合成结构化证据处理。VISOR采用单代理框架，通过结构化证据空间和视觉动作评估机制实现渐进式跨页推理。

3. **知识图谱与结构化知识 RAG**：如EHRAG构建超图同时捕获结构和语义关系，使用混合检索机制，在多个数据集上表现出色。Neuro-Symbolic框架将临床指南翻译为符号逻辑，通过SAT求解器验证推荐的一致性。

4. **多模态与视觉 RAG**：如AeroRAG将图像转换为结构化视觉知识，通过语义块检索构建提示，适用于航空视觉推理。KIRA是一个五阶段框架，解决专业视觉领域的关键问题，包括分块、编码和检索。

5. **评测、基准与诊断**：如MathNet构建了30,676个专家数学问题数据集，跨47国、17种语言，为数学推理和检索评估提供基准。FRESCO引入时间动态上下文评估重排序器，测试其对近期知识的优先级处理能力。

各类方法在不同场景下各有优势，复杂度从中等到高等不等，数据驱动程度较高，但在实际部署中的效率和可靠性方面仍需进一步验证。

# 趋势分析 Trend Analysis

根据307篇论文的分析，RAG研究呈现以下趋势：

1. **研究方向分化**：研究从通用检索管道向专门化变体发展，包括评估、领域适应、知识图谱或结构化知识、多模态设置以及可靠性导向设计。核心检索与排序方法成为最大类别（62篇），表明基础方法的持续重要性。

2. **方法模块化**：方法层日益模块化，检索器、重排序器、生成器、评估器，有时还包括代理或工具组件，被视为可分离的优化目标。

3. **评估与基准重要性**：评估和基准成为核心压力点，因为许多摘要强调性能声明，但并非所有摘要都充分暴露指标。

4. **数据驱动研究主导**：258篇论文明确采用数据驱动方法，表明实证研究的普及，但仍有42篇未指定数据集，119篇未指定指标。
